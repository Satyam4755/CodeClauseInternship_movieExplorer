About This Project

This Movie Explorer project was created as part of a task assigned by CodeClause. The goal was to build a clean, functional movie search interface using The Movie Database (TMDB) API. It allows users to search for movies, explore trending films, and view detailed information about each title in a smooth and engaging interface.

This project was completed in approximately 6 hours and is submitted under the Golden Level category. It showcases real-time API integration, responsive design, modal-based details view, and overall beginner-to-intermediate frontend development skills.
